Creating a Repository on GitHUB


1) Create New Repository



Click on the "+" sign on your github account next to "bell" notification icon and then click on create new repository.


2) Naming and scope of Repo



Enter the name of your repository which you want to set and then choose public or private you want to define the scope of repository


Git commands on terminal


1) git init



After creating empty folder in your pc,right click and select open with visual studio (or any other IDE) the create a file. After writing this command in terminal, a new git repository will be created in the folder.


2) git status



"git status" command displays the state of the working directory and the staging area. It lets you see which changes have been staged, which haven't, and which files aren't being tracked by Git.
